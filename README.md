# Nagoya Gluten-Free Map v2

名古屋グルテンフリーマップ - クリーンリビルド版

## 技術スタック
- Frontend: HTML5, CSS3, Vanilla JavaScript
- Map: Leaflet.js
- Backend: Supabase
- Authentication: Supabase Auth
- Hosting: GitHub Pages
